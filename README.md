# VUE2.0开发

```
npm run dev 临时开发
npm run build:prod 发布测试（提交dev分支前先打包测试）
```

1. 开发测试，编写代码
2. 主题风格符合本网站，代码规范，不要乱写，要条理清晰
3. 只需提交dev分支即可（提交dev分支前先打包测试）

# 开发思路

* 纯前端开发，参考已有网站
* 先找开源库
* 如果已有功能在其基础上开发
